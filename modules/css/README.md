# Basic CSS
## В данном модуле студентам необходимо:
Познакомиться с основами CSS:
- подключение стилей
- селекторы (специфичность, каскадность, наследование)
- стили для текста, шрифты
- единицы измерения
- фон, изображения в CSS
- блочная модель
